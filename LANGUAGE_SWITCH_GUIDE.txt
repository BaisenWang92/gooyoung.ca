【更新说明】
已修复语言切换问题！现在是完全纯静态网页实现，无需依赖外部文件加载。

【如何测试】
方法 1：直接在浏览器中打开文件
  - 在 Windows 资源管理器中，导航到：
    C:\Users\baise\Documents\Gooyoung\NewWebSite\bizpro1-master
  - 右键点击 index.html，选择"用浏览器打开" 或 "Open with Browser"
  - 在右上角导航菜单中找到 "EN" 和 "中文" 按钮
  - 点击测试语言切换

方法 2：使用测试文件
  - 打开 test-language.html 文件（同目录）
  - 这是一个简化版本，专门用于测试语言切换功能
  - 所有内容会实时翻译

【功能说明】
✓ 纯 HTML/CSS/JavaScript 实现
✓ 无需服务器，无需外部依赖
✓ 翻译数据嵌入在 i18n.js 中
✓ 用户语言偏好保存在浏览器本地存储（localStorage）
✓ 刷新页面后记住用户选择的语言

【翻译覆盖范围】
- 导航菜单：HOME, ABOUT, SERVICES, PORTFOLIO, TEAM, SKILLS, CLIENTS, PRICING, BLOG, CONTACT
- 关于部分标题和描述
- 服务部分标题和描述
- 联系我们部分标题和描述

【扩展翻译】
如需添加更多翻译内容，编辑 js/i18n.js 文件，在 translations 对象中添加新的键值对。

示例：
  "zh": {
    "hero": {
      "title": "欢迎来到古鹰网络",
      "subtitle": "专业的网站开发解决方案"
    }
  }

然后在 HTML 中添加 data-i18n 属性：
  <h1 data-i18n="hero.title">Welcome to Gooyoung</h1>

【故障排除】
如果文本没有改变，请：
1. 按 F12 打开浏览器开发者工具
2. 查看 Console 标签是否有错误信息
3. 确保 i18n.js 文件在 js 文件夹中
4. 清除浏览器缓存（Ctrl+Shift+Delete）并重新加载页面

【技术细节】
- 文件：js/i18n.js（包含所有翻译和逻辑）
- 方法：DOM 查询 data-i18n 属性 → 获取翻译 → 更新文本内容
- 存储：localStorage 保存 'preferredLang' 键
- 兼容性：所有现代浏览器支持（Chrome, Firefox, Safari, Edge）
