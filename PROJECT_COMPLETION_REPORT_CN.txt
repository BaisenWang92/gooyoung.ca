多语言系统架构:
─────────────
1. HTML标记:
   <a href="#home" data-i18n="nav.home">HOME</a>

2. JSON数据结构:
   {
     "en": { "nav": { "home": "HOME" } },
     "zh": { "nav": { "home": "首页" } }
   }

3. JavaScript逻辑:
   switchLanguage('zh') → 读取翻译 → 更新DOM → 保存preference

4. 浏览器存储:
   localStorage['preferredLang'] = 'zh'

技术栈:
──────
- HTML5 (结构层)
- CSS3 (样式层)
- Vanilla JavaScript (动态功能)
- JSON (数据存储)
- localStorage (客户端存储)

Q1: 语言切换不工作，怎么办？
A: 打开浏览器开发者工具 (F12)，检查控制台是否有红色错误信息。
   确保 i18n.js 和 i18n.json 文件存在且路径正确。

Q2: 怎样修改网站上的文字？
A: 编辑 js/i18n.json 文件，修改英文和中文部分的文本，
   然后刷新浏览器即可看到效果。

Q3: 如何添加第三种语言？
A: 在 i18n.json 中添加新的语言对象（如 "ja" 对日文），
   然后在 i18n.js 中更新 switchLanguage() 函数。

Q4: Logo 没有显示，怎么办？
A: 检查 ../Logo/logo.png 文件是否存在。
   如果文件不在该位置，请更新 index.html 中的引用路径。

Q5: 如何让用户访问时显示中文而不是英文？
A: 在浏览器控制台运行: localStorage.setItem('preferredLang', 'zh');
   然后刷新页面。

Q6: 可以在动态添加的内容中使用翻译吗？
A: 可以。使用 JavaScript 调用: 
   element.textContent = getTranslation('key.name');
